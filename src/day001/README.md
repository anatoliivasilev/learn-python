**Цель занятия:** Понять запуск Python-файла, `print`, переменные, литералы и чтение простого traceback.

**Источники:**
- **Теория/курс:** [CS50P Functions and Variables](https://cs50.harvard.edu/python/notes/0/)
- **Документация:** [Python: informal introduction](https://docs.python.org/3/tutorial/introduction.html)

**Блок 1 — Python 70 мин: установка, запуск, `print`, переменные и базовые литералы.**
- [ ] Установить Python 3; выполнить `python --version` или `python3 --version`.
  - `brew install python3`
- [ ] Создать `day01/main.py`; вывести три строки через `print`.
- [ ] Создать переменные `name`, `age`, `height`, `is_employed`, `middle_name`; вывести значение и `type`.
- [ ] Вычислить только выражения с литералами: `2+3`, `10-4`, `6*7`, `8/2`; вывести результат.
- [ ] Намеренно получить `SyntaxError` и `NameError`; в `notes.md` указать файл, строку и причину.

- [ ] Упражнение 1: файл выводит визитную карточку из заранее заданных переменных.

- [ ] Упражнение 3: с `minutes=135` вывести полные часы и остаток минут, используя `//` и `%` после примера из теории.
